Geschützte Seite erstellen

Läuft komplett in diesem Browser. Kein Server, kein Konto, kein Cloudflare-Secret nötig.

Falls Kopieren nicht klappt: Tippe ins Feld, halte gedrückt → "Alles auswählen" → "Kopieren". Füge den Text danach z.B. bei GitHub in eine neue Datei ein (z.B. index.html).

So geht's weiter:
1. Inhalt einfügen, Passwort wählen, Datei erzeugen.
2. Die heruntergeladene geschuetzte-seite.html irgendwo hochladen – Cloudflare Pages, Netlify, GitHub Pages oder eigener Webspace, ganz normal als statische Datei.
3. Wer die Seite öffnet, muss das Passwort eingeben. Ohne Passwort ist der Inhalt auch im Quelltext nicht lesbar – er ist verschlüsselt, nicht nur versteckt.

Ehrlich gesagt: Das schützt zuverlässig vor normalem Mitlesen und neugierigen Blicken. Es ist aber kein Schutz auf Bankniveau – mit sehr viel Rechenleistung liesse sich ein schwaches Passwort offline knacken. Nimm darum lieber eine kurze Wortfolge als ein einzelnes Wort.
